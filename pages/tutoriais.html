Tutoriais para Configuração

Aprenda a configurar assistentes de voz e dispositivos de baixo custo para auxiliar no cuidado do idoso.

Configurando a Alexa para lembretes de medicamentos

Passo a passo rápido:

  1. Por voz: "Alexa, lembre-me de tomar o remédio da pressão às 8 horas da manhã todos os dias"
  2. Pelo app: Abra o app Alexa > Mais > Lembretes > Adicionar Lembrete
  3. Skill Meus Medicamentos: Ative a skill para maior controle de doses
  4. Drop In: Configure para permitir que familiares façam chamadas diretas
Saiba mais no site oficial

Configurando Drop In para emergências

Como configurar:

  1. Abra o app Alexa → Dispositivos → Echo e Alexa
  2. Selecione o dispositivo da pessoa que precisa de cuidados
  3. Vá em Comunicação → Drop In → Permitir
  4. O cuidador pode dizer: "Alexa, faça Drop In em [nome do dispositivo]"
O Drop In funciona como um interfone: o dispositivo atende automaticamente.

Configurando Google Assistente para lembretes

Comandos úteis:

  • "Hey Google, me lembre de tomar remédio às 20h"
  • "Hey Google, crie um lembrete diário para verificar a pressão às 9h"
  • "Hey Google, faça uma chamada para [contato de emergência]"

Dica: O Google Assistente também permite criar rotinas personalizadas para o dia a dia do idoso.
